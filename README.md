# Documentação do trabalho de Desenvolvimento Web

## Visão Geral
Trabalho da disciplina de Desenvolvimento Web, tem como ideia ser um blog sobre o curso de ADS, apresentado informações úteis sobre o curso. Criado por Otávio de Moraes e Arthur Mauro

## Dependências e Bibliotecas
- **Chart.js**: Biblioteca para criação de gráficos. [Documentação](https://www.npmjs.com/package/chart.js)

## Funcionalidades Principais
- **Tema claro/escuro**: Alterar entre tema claro ou escuro.
- **Tamanho de fonte**: Aumentar ou diminuir texto do corpo principal da página.
- **Cookies**: Uso de cookies para armazenar as preferências do usuário.
